Question 1:

In how many category processes is Artificial Intelligence classified in?

a) 4
b) 5
c) 2
d) 3

The correct answer is d) 3: Sensing, Reasoning, and Acting.

Question 2:

Which of the following is the common language for Artificial Intelligence?

a) Python
b) Java
c) Lisp
d) PHP

The correct answer is b) a) Python.

Question 3:

The proposition symbols in AI are?

a) true and false
b) true, false and null
c) true
d) false

The correct answer is a) true and false.

Question 4:

How does an AI agent interacts with its environment?

a) Using sensors and perceivers
b) Using only sensors
c) Using only perceivers
d) None

The correct answer is a) Using sensors and perceivers.

Question 5:

Which of the following symbols in AI are logical symbols?

a) Negation
b) Conjunction
c) Implication
d) All of the above

The correct answer is d) All of the above.
