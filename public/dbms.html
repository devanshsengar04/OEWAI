Question 1:

WWhat is DBMS?

a) DBMS is a collection of queries
b) DBMS is a high-level language
c) DBMS is a programming language
d) DBMS stores, modifies and retrieves data

The correct answer is d) DBMS stores, modifies and retrieves data.

Question 2:

Which type of data can be stored in the database?

a) Image oriented data
b) Text, files containing data
c) Data in the form of audio or video
d) All of the above

The correct answer is d) All of the above.

Question 4:

Which of the following provides the ability to query information from the database and insert tuples into, delete tuples from, and modify tuples in the database?

a) DML(Data Manipulation Language)
b) DDL(Data Definition Language)
c) Query
d) Relational Schema

The correct answer is a) DML(Data Manipulation Language).

Question 5:

______________ is a set of one or more attributes taken collectively to uniquely identify a record.

a) Primary Key
b) Foreign key
c) Super key
d) Candidate key

The correct answer is c) Super key.
