Question 1:

In Operating Systems, which of the following is/are CPU scheduling algorithms?

a) Priority
b) Round Robin
c) Shortest Job First
d) All of the mentioned

The correct answer is d) All of the mentioned.

Question 2:

In Operating Systems, which of the following is/are CPU scheduling algorithms?

a) lack of paper in printer
b) connection failure in the network
c) power failure
d) All of the mentioned

The correct answer is d) All of the mentioned.

Question 3:

Which of the type of OS reads and reacts in terms of actual time?

a) Quick sharing OS
b) Time sharing OS
c) Real Time OS
d) Batch OS

The correct answer is c) Real Time OS.

Question 4:

Which of the following schedules threads?

a) Virtual Memory
b) Operating System
c) CPU
d) Input

The correct answer is b) Operating System.

Question 5:

The main memory accommodates ____________?

a) cpu
b) user processes
c) operating system
d) all of the mentioned

The correct answer is c) operating system.
