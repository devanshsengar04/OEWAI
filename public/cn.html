Question 1:

The full form of OSI is?

a) Operating System Interface
b) Operating System Interconnection
c) Operating System Internet
d) Open System Interconnection

The correct answer is d) Open System Interconnection.

Question 2:

What is the number of layers in the OSI model?

a) 2 layers
b) 4 layers
c) 7 layers
d) 9 layers

The correct answer is c) 7 layers.

Question 3:

Choose among the following, which is the most common internet protocol.

a) PPP
b) FTP
c) TCP/IP
d) SMTP

The correct answer is d) SMTP.

Question 4:

What does TCP/IP stand for?

a) Telephone control protocol/Internet protocol.
b) Transmission control protocol/Internet protocol.
c) Transmission control protocol/International protocol.
d) None

The correct answer is b) Transmission control protocol/Internet protocol..

Question 5:

Identify the layer which provides service to the user.

a) Session layer
b) Application layer
c) Presentation layer
d) Physical

The correct answer is b) Application layer.
