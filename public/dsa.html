Question 1:

Which of the following is a linear data structure?

a) Array
b) AVL trees
c) Binary trees
d) Graphs

The correct answer is a) Array.

Question 2:

Which of the following is not the type of queue?

a) Priority Queue
b) Single Ended Queue
c) Circular Queue
d) Ordinary Queue

The correct answer is b) Single Ended Queue.

Question 3:

When a pop() operation is called on an empty queue, what is the condition called?

a) Overflow
b) Underflow
c) Syntax error
d) Garbage value

The correct answer is b) Underflow.

Question 4:

Which of the following data structures can be used to implement queues?

a) Stack
b) Arrays
c) Linked Lists
d) All of the above

The correct answer is d) All of the above.

Question 5:

Which of the following data structures finds its use in recursion?

a) Stack
b) Arrays
c) Linked Lists
d) Queues

The correct answer is a) Stack.
